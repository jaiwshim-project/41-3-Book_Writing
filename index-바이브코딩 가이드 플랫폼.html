Claude

Claude 소개

Claude는 Anthropic이 개발한 AI 어시스턴트로, 4가지 방식으로 사용할 수 있습니다.

Claude 4종 비교

종류 접근 방식 특징
모바일 앱 iOS / Android 이동 중 간편한 대화
웹 claude.ai 브라우저에서 바로 사용
데스크톱 앱 Windows / Mac 독립 실행 앱
Claude Code 터미널 (CLI) 코드 작성 & 파일 편집 가능

바이브 코딩에서 핵심은 Claude Code입니다. 다른 3종은 대화만 가능하지만, Claude Code는 실제로 파일을 만들고, 코드를 작성하고, 명령어를 실행할 수 있습니다.

설치

Claude Code 설치 가이드

7단계로 Claude Code를 설치하고 실행해봅시다.

1단계: Node.js 설치

Claude Code는 Node.js 위에서 실행됩니다.

winget install OpenJS.NodeJS.LTS

설치 확인:

node -v && npm -v

2단계: Git 설치

코드 버전 관리에 필요합니다.

winget install Git.Git

3단계: Claude Code 설치

npm으로 전역 설치합니다.

npm install -g @anthropic-ai/claude-code

4단계: 최초 실행

프로젝트 폴더에서 claude를 실행합니다.

mkdir my-project && cd my-project
claude

5단계: Anthropic 계정 인증

처음 실행하면 브라우저가 열리며 로그인을 요청합니다. Anthropic 계정이 없다면 회원가입 후 진행합니다.

6단계: API 결제 설정

Claude Code는 API 사용량에 따라 과금됩니다. console.anthropic.com에서 결제 수단을 등록합니다.

사용료 체계

Claude Code의 과금 방식과 비용을 알아봅시다.

API 과금 방식

Claude Code는 API 호출 시 토큰 단위로 과금됩니다.

모델 입력 (1M 토큰) 출력 (1M 토큰)
Sonnet 4.5 $3 $15
Opus 4 $15 $75

참고: 일반적인 바이브 코딩 세션(1~2시간)에서 약 $1~5 정도가 소요됩니다. 복잡한 작업일수록 토큰 사용량이 늘어납니다.

팁

팁 & 주의사항

Claude Code를 효과적으로 사용하기 위한 팁입니다.

효과적인 프롬프트 작성법

  • 구체적으로 말하기 — "웹사이트 만들어줘" 보다 "왼쪽 사이드바, 오른쪽 본문 레이아웃의 다크 모드 웹사이트를 만들어줘"가 낫다
  • 한 번에 하나씩 — 여러 작업을 한꺼번에 요청하기보다 하나씩 요청하고 확인하기
  • 자주 커밋하기 — 마음에 드는 결과가 나오면 바로 git 커밋

주의사항

  • API 키나 비밀번호 같은 민감 정보를 프롬프트에 직접 넣지 말 것
  • 대화가 길어지면 /compact로 요약하여 토큰 절감
  • 결과물은 항상 직접 확인하고 테스트할 것

토큰 절감 방안

Claude Code 사용 비용을 줄이는 실용적인 방법입니다.

/compact 명령어 활용

대화가 길어지면 /compact 명령어로 대화 내용을 요약합니다. 컨텍스트 길이가 줄어들어 토큰 사용량이 감소합니다.

명확하고 구체적인 프롬프트

모호한 요청은 AI가 여러 번 시도하게 만들어 토큰을 낭비합니다. 처음부터 구체적으로 요청하면 한 번에 원하는 결과를 얻을 수 있습니다.

CLAUDE.md 파일 활용

프로젝트 루트에 CLAUDE.md를 작성하면 매번 프로젝트 설명을 반복하지 않아도 됩니다.

/cost로 비용 확인

/cost 명령어로 현재 세션의 사용 비용을 수시로 확인하세요.

GitHub

GitHub 소개 및 설정

GitHub는 1억 명 이상의 개발자가 사용하는 세계 최대의 코드 호스팅 플랫폼입니다.

GitHub란?

GitHub는 Git 기반의 코드 호스팅 및 협업 플랫폼입니다. 2008년 설립되어 현재 Microsoft 소속입니다.

  • Repositories(저장소) — 프로젝트의 코드, 파일, 이력을 보관하는 공간
  • Issues — 버그 신고, 기능 요청, 작업 관리를 위한 트래커
  • Pull Requests — 코드 변경을 제안하고 리뷰받는 협업 도구
  • Actions — 테스트, 빌드, 배포를 자동화하는 CI/CD
  • Pages — 저장소에서 직접 정적 웹사이트 호스팅

계정 생성

  • github.com 접속 → "Sign up" 클릭
  • 이메일, 비밀번호, 사용자명 입력
  • Free 플랜으로 충분 (무제한 공개/비공개 저장소)

Git 초기 설정

GitHub 가입 후 로컬 Git에 사용자 정보를 등록합니다. 이 정보가 커밋 기록에 표시됩니다.

git config --global user.name "내 이름"
git config --global user.email "GitHub 가입 이메일"

설정 확인:

git config --list

SSH 키 설정 (선택): HTTPS 대신 SSH로 연결하면 매번 비밀번호를 입력하지 않아도 됩니다. GitHub 공식 문서의 "Connecting to GitHub with SSH" 가이드를 참고하세요.

Supabase

Supabase 소개 및 설정

Supabase는 백엔드 서비스를 제공하는 플랫폼입니다. 데이터베이스, 인증, 파일 저장소를 클릭 몇 번으로 사용할 수 있습니다.

Supabase란?

Firebase의 오픈소스 대안으로, PostgreSQL 기반의 강력한 백엔드 서비스입니다.

  • 데이터베이스 — PostgreSQL 기반, SQL 지원
  • 인증(Auth) — 이메일, 소셜 로그인 지원
  • 스토리지 — 이미지, 파일 업로드/관리
  • 실시간 — 데이터 변경 시 자동 알림

시작하기

supabase.com에서 회원가입 후 새 프로젝트를 생성합니다.

npm install @supabase/supabase-js
Supabase

데이터베이스

Supabase의 PostgreSQL 데이터베이스를 사용하는 방법입니다.

테이블 만들기

Supabase 대시보드의 Table Editor에서 GUI로 테이블을 만들 수 있습니다. 또는 SQL로 직접 생성할 수도 있습니다.

CREATE TABLE posts (
id BIGINT GENERATED ALWAYS AS IDENTITY PRIMARY KEY,
title TEXT NOT NULL,
content TEXT,
created_at TIMESTAMPTZ DEFAULT NOW()
);

데이터 조회 (JavaScript)

const { data, error } = await supabase
.from('posts')
.select('*')
Supabase

인증(Auth)

Supabase의 인증 기능으로 회원가입, 로그인을 구현할 수 있습니다.

이메일 회원가입

const { data, error } = await supabase.auth.signUp({
email: 'user@example.com',
password: 'password123'
})

지원하는 로그인 방식

  • 이메일 / 비밀번호
  • Google, GitHub, Apple 소셜 로그인
  • 매직 링크 (비밀번호 없이 이메일로 로그인)
  • 전화번호 인증
Supabase

스토리지

이미지, 동영상, 파일을 업로드하고 관리하는 스토리지 서비스입니다.

버킷 생성

Supabase 대시보드 → Storage에서 새 버킷을 생성합니다. Public(공개) 또는 Private(비공개)를 선택할 수 있습니다.

파일 업로드

const { data, error } = await supabase.storage
.from('images')
.upload('photo.jpg', file)
Supabase

RLS & 보안

Row Level Security로 데이터 접근을 제어합니다.

RLS란?

Row Level Security(행 수준 보안)는 데이터베이스 테이블의 각 행에 대해 누가 읽고, 쓰고, 수정할 수 있는지를 제어하는 PostgreSQL 기능입니다.

RLS 정책 예시

-- 본인의 데이터만 조회 가능
CREATE POLICY "Users can view own data"
ON posts FOR SELECT
USING (auth.uid() = user_id);
Supabase

벡터 DB & AI

pgvector 확장을 활용한 AI 임베딩 저장 및 유사도 검색입니다.

벡터 DB란?

텍스트, 이미지 등을 숫자 벡터로 변환하여 저장하고, 유사한 항목을 빠르게 검색하는 데이터베이스입니다. AI 기반 검색, 추천 시스템에 활용됩니다.

pgvector 활성화

-- Supabase SQL Editor에서 실행
CREATE EXTENSION IF NOT EXISTS vector;
Vercel

Vercel 소개 및 배포

Vercel은 웹사이트를 인터넷에 공개하는 배포 플랫폼입니다. GitHub와 연결하면 코드를 올릴 때마다 자동으로 배포됩니다.

Vercel이란?

웹사이트를 전 세계 누구나 접속할 수 있게 만들어주는 서비스입니다.

  • GitHub 저장소 연결 → 자동 배포
  • 무료 플랜으로 개인 프로젝트 충분
  • HTTPS(보안 연결) 자동 적용
  • 전 세계 CDN으로 빠른 로딩 속도

배포 방법

npm install -g vercel
vercel login
vercel --prod

또는 vercel.com에서 GitHub 저장소를 연결하면 끝입니다.

VS Code

VS Code 소개 및 설치

VS Code는 Microsoft가 만든 무료 코드 편집기입니다. 바이브 코딩의 모든 작업이 여기서 이루어집니다.

왜 VS Code인가?

  • 무료이면서 강력한 기능
  • 내장 터미널에서 Claude Code 실행 가능
  • 수천 개의 확장 프로그램
  • Git 연동 기본 지원

설치

winget install Microsoft.VisualStudioCode

또는 code.visualstudio.com에서 다운로드합니다.

VS Code

UI 구조 & 편집

VS Code 화면 구성과 편집 기능을 알아봅시다.

화면 구성

  • 액티비티 바 — 왼쪽 아이콘 모음 (탐색기, 검색, Git, 확장 등)
  • 사이드바 — 파일 탐색기, 검색 결과 등 표시
  • 에디터 — 코드를 편집하는 중앙 영역
  • 패널 — 하단의 터미널, 출력, 문제 탭
  • 상태 바 — 맨 아래 줄. 현재 파일 정보 표시
VS Code

Git & 소스 제어

VS Code 내장 Git 기능으로 버전 관리를 합니다.

소스 제어 패널

왼쪽 액티비티 바에서 분기 아이콘을 클릭하면 소스 제어 패널이 열립니다. 변경된 파일 목록, 스테이징, 커밋을 GUI로 처리할 수 있습니다.

주요 기능

  • 변경 사항 시각적 비교 (diff)
  • 파일별 스테이징 / 언스테이징
  • 커밋 메시지 입력 및 커밋
  • 브랜치 생성, 전환, 병합
  • 원격 저장소 push / pull
VS Code

Vibe Coding Manual이란?

Vibe Coding Manual은 현대 웹 개발에 필요한 5가지 핵심 도구(Claude Code, GitHub, Supabase, Vercel, VS Code)의 사용법을 한곳에 모은 종합 바이브코딩 완벽 매뉴얼입니다. 초보자부터 실무 개발자까지 단계별로 학습할 수 있도록 구성되어 있습니다.

사용 방법

  • 상단 내비게이션 — 원하는 도구(Claude, GitHub, Supabase, Vercel, VS Code)를 클릭하면 해당 가이드의 첫 페이지로 이동합니다.
  • 사이드바 메뉴 — 좌측 사이드바에서 각 도구의 세부 항목을 선택하여 원하는 내용으로 바로 이동할 수 있습니다.
  • 코드 복사 — 코드 블록 우측 상단의 "복사" 버튼을 클릭하면 명령어를 클립보드에 복사합니다.
  • 랜딩 페이지 — 좌측 상단의 "Vibe Coding Manual" 로고를 클릭하면 홈 화면으로 돌아갑니다.

권장 학습 순서

  1. VS Code 설치 — 코드 편집기가 모든 개발의 시작점입니다. winget install Microsoft.VisualStudioCode
  2. Node.js 설치 — JavaScript 런타임과 npm을 설치합니다. winget install OpenJS.NodeJS.LTS
  3. Git & GitHub 설정 — 버전 관리 환경을 구축합니다. winget install Git.Git
  4. Claude Code 설치 — AI 코딩 어시스턴트를 활용합니다. npm install -g @anthropic-ai/claude-code
  5. Supabase & Vercel — 백엔드와 배포 환경을 설정합니다. npm install -g vercel
